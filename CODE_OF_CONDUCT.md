# Code of Conduct

## Our Commitment

We are committed to a friendly, inclusive, and harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Focus feedback on ideas and code, not people.
- Welcome newcomers and different viewpoints.

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discrimination of any kind
- Trolling, deliberate intimidation, or hostile behavior
- Publishing private information without consent

## Enforcement

Project maintainers may edit, remove, or reject comments, issues, or pull requests that violate this Code of Conduct.

## Reporting

Report behavior concerns by opening a private security advisory or by contacting the maintainer.

## Scope

This Code of Conduct applies in all project spaces, including issues, discussions, pull requests, and release comments.

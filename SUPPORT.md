# Support

## Getting Help

Use GitHub Discussions for usage and setup questions.

Use GitHub Issues for reproducible bugs and feature requests.

## Before Opening an Issue

- Confirm you are on the latest release
- Include platform details (Windows/Android, version, device)
- Include logs or screenshots when possible

## Maintainer Scope

Best-effort support is provided for:

- Build and setup issues
- Runtime bugs
- Release packaging issues

No guaranteed SLA is provided.
